To use CipherMaster on platforms other than windows you need to follow the following steps:

1. Install python 3 from https://www.python.org/downloads/
2. Open your Terminal and run pip install pyperclip to install the dependencies
3. [ONLY ON LINUX] 
	- On Debian/Ubuntu-based systems: sudo apt install xclip or sudo apt install xsel
	- On Fedora: sudo dnf install xclip
4. Run CipherMaster.py with the Python 3 app

note: You can build your own platform based executable using pyinstaller https://pyinstaller.org/en/stable/
